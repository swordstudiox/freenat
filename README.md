# 免费frp内网穿透服务器

##一、美国frp线路2：0.16.1版本（兼容0.10.0-0.16.1版本）
*连接地址（server_addr）为：freenat.bid
*通讯端口（server_port）为：7000
*特权认证密码(privilege_token)为：frp888
*状态查询：http://freenat.bid:7500
*查询帐号和密码均为：admin
*默认http端口为80
*默认https端口为443
*端口全部开放
*提供域名（custom_domains）：*.freenat.bid(*自定义)
*支持KCP协议
*支持P2P穿透（xtcp）
*本服务器长期服务
