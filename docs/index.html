关于

1、frp是一个高性能的反向代理/端口映射应用，可以帮助您轻松地进行内网穿透，对外网提供服务， 支持tcp, udp, http, https等协议类型，并且支持根据域名进行路由转发web服务。
2、frp内网穿透主要给没有公网IP的用户，实现远程桌面、远程控制路由器、 外网访问在内网搭建的WEB、FTP、SMB服务器、远程查看摄像头、调试一些远程的API（比如微信公众号，企业号的开发）等。
